# devcenter-retirees
a full-stack application that logs into a medical application.

> Register.

> Login.

## Built with React, TypeScript, Tailwind and NestJS

## Install

```sh
cd .\backend\
cd .\frontend\
npm install
```

## Usage

```sh
cd .\backend\
npm start
cd .\frontend\
npm start
```

Spins up local servers on port 3000 and 4000


Example:

```sh
Local:     http://localhost:3000
```
